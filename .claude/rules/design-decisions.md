# 設計上の重要な判断

| 判断 | 理由 |
|------|------|
| 静的HTML（フレームワークなし） | ページ数が少なく（5ページ）、内容の更新頻度も低い。ビルド不要でシンプル |
| インラインCSS | Claude Designからの変換。ページ数が少ないため外部CSS化の優先度は低い |
| Jekyll除外を活用 | `.claude/` と `_reference/` はJekyllが配信しない。非公開コンテキストの置き場 |
| CNAME でカスタムドメイン | nukanoba.co.jp。DNSはお名前.comで管理 |
| 画像は /assets/images/ | 全ページ共通パス |
| 実在情報は推測で補完しない | 電話番号、住所、メール、LINE URL等はJSONの正本データに明記がない限り推測禁止 |
