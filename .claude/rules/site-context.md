## 伊藤大助プロジェクトの全体像

### プロジェクト概要

伊藤大助さん（クラムボン ドラマー / 尚美ミュージックカレッジ専門学校 ドラム講師）の独立収益化を支援するプロジェクト。NUKANOBAの自社運営プロジェクトとして位置づけ。

### 展開ロードマップ

YouTube → LINE Business → 有料リズム教育アプリ（現在はYouTube成長期〜LINE収益化初期）

### YouTubeチャンネル（@daisukeitoh_drums）

- 2026年2月開設。長尺動画（ドラム相談・概念解説）＋ショート動画（毎日基礎練シリーズ）の2軸
- 2026年6月時点: 登録者約1,800人、総再生46万回、公開動画76本（長尺22本＋ショート54本）
- ショートの「毎日基礎練」シリーズが主力。特に16分音符アクセント系・フラム系が再生数を牽引
- 長尺は「プロの視点」「リズム概念」カテゴリが中心。1本あたり1,000〜2,300再生程度

### コンテンツピラー

1. **基礎技術** — グリップ、ストローク、フットワーク等
2. **リズム概念** — ポリリズム、タイム感、グルーヴ理論
3. **プロの視点** — 25年以上の経験からのインサイト
4. **曲解説** — 特定の曲のドラムパート分析

### LINE公式アカウント

- アカウント名: 伊藤大助のドラム・リズム相談室
- 2026年6月13日運用開始
- 役割: 無料PDF配布（基礎練ブック）、個人レッスン相談・予約、イベント案内
- LP上の全CTAはLINEに誘導する設計

### 無料PDF「基礎練ブック」

- ショート動画49本を7ジャンルに整理した譜面集
- LINE友だち追加で配布。LPのFREE PDFセクションで訴求

### 収益化（現在の状態）

- 対面個人レッスン: 9,800円（税込）/ 1時間。オンライン決済リンクで事前決済
- イベント: ドラムセミナー・交流イベントを企画中（Coming Soon）
- オンラインレッスン: 将来対応予定（現時点ではLP・特商法に記載のみ）

### ターゲットオーディエンス

- クラムボンのファン

### 管理リポジトリ

YouTubeチャンネルの運営データ・戦略・ナレッジは ssossan/rhythm-platform（プライベート）で管理。動画メタデータ、アナリティクス、文字起こし、譜面パイプライン、コンテンツカレンダー、市場分析等を格納。

### 正本データの所在

会社情報・サービス情報・特商法・問い合わせに関するJSONは **このリポジトリの `_reference/` が正本**。

| データ | 正本ファイル |
|--------|-------------|
| 会社情報 | _reference/company.json |
| サービス情報 | _reference/service.json |
| 特商法 | _reference/legal.json |
| 問い合わせ | _reference/contact.json |

- HTMLの情報を更新する際は `_reference/` のJSONを先に更新し、その内容をHTMLに反映する
- rhythm-platform にはこれらのJSONを持たない（YouTube運営に専念）
